# 📱 Test Calm Awareness App on Your Phone

## 🚀 **Instant Testing Options**

### **Option 1: Expo Snack (Easiest)**
1. Go to **snack.expo.dev** in your web browser
2. Copy and paste the entire App.js code from this project
3. Click "Save" to create a Snack
4. Open **Expo Go** app on your phone
5. Scan the QR code that appears on the Snack website
6. The app will load directly on your phone!

### **Option 2: Local Development Server**
1. **Install Expo Go** on your phone (App Store/Play Store)
2. **Download this project** to your computer
3. **Run these commands:**
   ```bash
   cd calm-awareness-app
   npm install
   npm start
   ```
4. **Scan the QR code** that appears in your terminal/browser
5. The app loads on your phone through Expo Go

### **Option 3: Direct APK (Android Only)**
If you want to install the app directly:
1. Run: `expo build:android --type apk`
2. Download the generated APK file
3. Install it directly on your Android phone
